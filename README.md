
# Chatbot using chatgpt

A chatbot that uses the OpenAI api to fetch results.


## Run Locally

Clone the project

```bash
  git clone https://github.com/yashbijoor/AI-Chatbot.git
```

Go to the project directory

```bash
  cd AI-Chatbot
```

Install dependencies

```bash
  python -r requirements.txt
```
Run server
```bash
  flask run
```
